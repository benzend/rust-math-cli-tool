# rust-math-cli-tool

A simple math CLI built with Rust

## Commands

cargo run -- maths "5 * 4 + 3 / 2 - 1"

cargo run -- add <int> <int> (only handles two args)

cargo run -- subtract <int> <int> (only handles two args)
